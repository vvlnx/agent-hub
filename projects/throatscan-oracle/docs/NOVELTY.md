# Dimension 4 — Novelty & Potential

## One-line pitch

ThroatScan is **not** a single LLM prompt or theme screener. It orchestrates parallel Agent Hub news/macro research, constrained supply-chain reasoning, Bitget tradability verification, and risk-managed simulation — with hard gates the LLM cannot bypass.

## Why only AI Agents can do this

| Traditional screener / chat | ThroatScan Agent workflow |
| --- | --- |
| Sequential Q&A | Parallel MCP research while reasoning runs |
| Popularity / theme ranking | Structural bottleneck thesis + rejected alternatives |
| Opaque scores | Bitget API evidence + SHA-256 export |
| LLM picks tickers | Deterministic engine + hard constraints |

## In-app evidence (after each run)

- **Agent Workflow** panel — 7 steps with skill names, `tools_used`, fetch timestamps
- **Hard constraints** — ticker allowlist, LLM no-pick, Bitget gate, capped overlay
- **Rebalance agent plan** — periodic rebalance + trailing stop triggers from backtest policy
- **Growth roadmap** — Phase 1 (live), Phase 2 (paper/demo), Phase 3 (event agents)

## Demo script (30 seconds for judges)
